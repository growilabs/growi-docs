# GROWI Commons Libraries

## [path-utils](util/path-utils.html)

An utility object provides functions working with GROWI page path.

Usage:
```javascript
// import
import { pathUtils } from 'growi-commons`;
// require
const { pathUtils } = require('growi-commons);
```
