# Insert links

## Using the insert link function on the toolbar

1. In edit mode, click the plus button on the toolbar and select "Link".
    <img   :src="$withBase('/assets/images/en/add_link.png')" alt="add_link">
1. The Edit Link dialog is opened. Enter the Link : `https://docs.growi.org/`, Label: `GROWI Docs` and click the "Done" button.

    <img :src="$withBase('/assets/images/en/edit_link.png')" alt="edit_link">
1. The link is inserted. Check it in the preview.

    <img :src="$withBase('/assets/images/en/link.png')" alt="link">

## Writing in Markdown

- Write in the edit screen as follows.

  ```
  [Label](URL or page path)
  ```
