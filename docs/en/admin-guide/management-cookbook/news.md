# News Delivery

The News Delivery feature, added in v7.5.4, allows users to receive news such as announcements and release information from the GROWI development team via in-app notifications.

## Types of News

The following types of news are delivered. The audience varies by type.

| Type | Audience |
| :--- | :--- |
| Version releases | Administrators |
| Security patches | Administrators |
| New feature introductions (Tips) | All users |
| Maintenance announcements | All users |
| Deprecation / EOL notices | Administrators |

## Settings

Administrators can enable or disable news delivery from `Admin Panel` > `App Settings` > `News delivery`.
