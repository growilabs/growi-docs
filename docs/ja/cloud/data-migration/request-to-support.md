# GROWI.cloud で作成された GROWI へのデータ移行 (サポートへの移行作業依頼)

## 概要

- ご自身の環境に構築したOSS版GROWI(オンプレミス版)からGROWI.cloudへデータを移行するには2種類の方法がございます。
    1. **GROWI.cloud サポートに移行を依頼する (本ページで解説)**
    1. データアーカイブ・インポート機能を利用する
- 移行元となる GROWI のサーバから取得していただいた DB の dump データおよび添付ファイルを弊社でお預かりし、GROWI.cloud 上でリストア作業をする方法です。
  - 有料対応となりますが、**移行元GROWIのバージョンによらず**ご利用いただけます。
  - 移行元 GROWI の環境によって必要な作業が異なるため、作業を**2段階**に分けて進めさせていただきます。

### 過去の対応実績

- **移行元 GROWIバージョン**
  - v4系、v5系など、v7系以前のバージョンからの移行
- **データ保存先**
  - MongoDB
    - GridFS
  - AWS

## 作業内容

### Phase-1 ：費用 11〜25万円(過去実績)、5〜10営業日

- Phase-1では基本的なデータ移行作業を実施いたします。
  - 受領したデータ (mongodump や添付ファイルなど) の検証
  - 受領した mongodump を GROWI.cloud へリストア
    - **v7以前のバージョンからの移行の場合は追加費用が発生します。**
    - v7以上のバージョンへのアップデートを事前に実施していただいた上でmongodumpデータを取得していただければ、追加費用は発生いたしません。
  - 添付ファイルの移行
    - **移行元GROWIのファイルアップロード先によっては追加費用が発生いたします。(別途お見積もり)**
  - data migration の実施
  - (必要な場合) お客様にて v5 互換形式への変換
- 移行元GROWIのバージョン、記法、データ保存先によっては、基本的なデータ移行作業だけでは完全にデータを移行できない場合がございます。
- Phase-1完了後、弊社・お客様の双方でデータ移行結果を確認し、Phase-2 の追加対応が必要かを確認させていただきます。

### Phase-2： 費用、作業期間ともに別途お見積もり

- Phase-1 完了後、実際のデータを確認した上で下記のような作業が必要と判断した場合はPhase-2のお見積もりをさせていただきます。
  - データの不整合修正
    - data migrations実施時のデータ不整合修正
    - v5互換形式への変換時のデータ不整合修正
    - その他のデータ不整合修正：都度見積
  - 本文の一括置換
  - URL・リンクの修正
  - 画像パスの変更
  - 特定文字列の置換
- その他カスタマイズ対応

## 注意点

### 現在利用中のGROWIバージョンがv5系以下の場合

- GROWI.cloudではv7以降のGROWI のみが提供されておりますので、移行先GROWIは自動的に最新バージョンにアップデートされます。
- **現在利用中のGROWIがv5系以下の場合、移行の際に追加作業・費用が発生いたします。**
- そのため、可能であれば、移行作業前にGROWIをv7以上のバーションにアップデートしていただいてからの移行依頼をおすすめしております。
  - [GROWI v5.0.x へのアップグレード](https://docs.growi.org/ja/admin-guide/upgrading/50x.html)
  - [GROWI v7.0.x へのアップグレード](https://docs.growi.org/ja/admin-guide/upgrading/70x.html)
- また、Phase-1移行作業後、お客様側でページのデータ形式を **v5互換形式**へと変換を行なっていただく必要がございます。変換方法については下記ページをご確認ください。
  - [新しいv5互換形式形式について](https://docs.growi.org/ja/admin-guide/upgrading/50x.html#%E6%96%B0%E3%81%97%E3%81%84-v5-%E4%BA%92%E6%8F%9B%E5%BD%A2%E5%BC%8F%E3%81%AB%E3%81%A4%E3%81%84%E3%81%A6)

### ページリンクについて

- ページの記載内容をそのまま移行するため、例えば移行元GROWIのURLがリンクとしてページ内に記載されていた場合はアクセスができなくなってしまいます。
- [[/Sandbox]] のようにページパスでリンクを指定していただけると、移行後も書き換えをせずにご利用いただけます。
  - [ページリンクを記述する](https://docs.growi.org/ja/guide/features/page_linker.html)
- ページ内容の書き換えについては弊社側では判断ができませんので、事前にお客様側でご対応いただけますと幸いです。

### GROWI.cloudの利用開始タイミングについて

- 移行先となるGROWI.cloudで管理されたGROWIは、**Phase-1の移行開始時までにお客様ご自身で作成**していただく必要がございます。
  - [無料トライアル](/ja/cloud/freetrial.html)で、管理者となるGROWI.cloudアカウント(組織)を作成
  - 移行先となる GROWI を作成
- そのため、移行開始と同時にGROWI.cloud の利用も開始となります。
  - 請求タイミングにつきましては支払い方法によって異なります。

### 【GROWI】Phase-1 見積もり

- 移行依頼フォーマットでのご回答内容を確認し、場合によっては追加でヒアリングをさせていただいた上でPhase-1のお見積もりを提示させていただきます。
- この時点で、Phase-2の発生可能性をご案内できる場合もございます。

### 【お客様】移行データ・移行先環境の準備

- お見積もりに問題がない場合、お客様側で下記データや情報をご準備いただき、弊社にご提供いただきます。なお、弊社側からお客様環境へアクセスしてのデータ取得作業は対応しておりません。
  - **移行元GROWIに関するもの**
    - GROWI の mongodump データ
    - 添付ファイルの実体 (必要に応じて)
    - PASSWORD_SEED 値 (設定している場合)
  - **移行先GROWIに関するもの (GROWI.cloud)**
    - 移行依頼フォーマットで未回答の場合
      - 契約プラン
      - 契約オプション
      - GROWIのURL
- 移行作業に際しまして、弊社としては特に契約書やNDA(秘密保持契約書)の締結を必須としておりません。契約書対応が必要であればご連絡ください。
  - NDAの締結が必要な場合は、恐れ入りますがお客様側からドラフトをご共有いただいております。

### 【GROWI】Phase-1 移行作業実施 (5〜10営業日)

- 全てのデータをいただいた翌日から5〜10営業日以内に、弊社側で移行作業を実施します。
  - 移行元GROWIのバーションやファイルアップロード先によっては特別対応が必要になるため、移行期間が上記より長くなる場合もございます。

### 【お客様】Phase-1 作業後確認

- お客様側でGROWIの表示に問題がないか、表示できていないファイルがないかをご確認いただきます。

### 【GROWI】Phase-2 見積もり

- お客様側の Phase-1 作業後確認の結果も踏まえ、Phase-2の作業が必要かを判断させていただきます。
- Phase-1で移行作業が終了する場合もございます。

### 【GROWI】Phase-2 作業実施

- 見積もりに問題がない場合、Phase-2の作業を実施させていただきます。
- 作業内容によって期間が異なります。

### 【お客様】Phase-2 作業後確認

- お客様側でGROWIの表示に問題がないか、表示できていないファイルがないかをご確認いただきます。
- 確認が完了し次第、移行作業に関する請求書を発行させていただきます。
  - 移行作業に関する費用についてはクレジットカード払いには対応しておりません。予めご了承ください。
