# GROWI.cloud で作成された GROWI へのデータ移行

## OSS版 GROWIから

- ご自身の環境に構築した OSS 版GROWI (オンプレミス版)から、GROWI.cloud で作成された GROWI へデータを移行する方法は2種類ございます。
    1. [**GROWI.cloud サポートに移行を依頼する (推奨)**](/ja/cloud/data-migration/request-to-support.html)
        - 移行元 GROWI のバージョンによらずご利用いただけます。
    1. [**データアーカイブ・インポート機能を利用する**](/ja/cloud/data-migration/data-archive-import.html)
        - 移行元 GROWI が v7.0.0 以降にアップデート済の場合、かつ上記のページに書かれた条件を満たしている場合のみご利用いただけます。

## GROWI.cloud で作成された GROWI から

- GROWI.cloud で作成された GROWI から、GROWI.cloud で作成された 別の GROWI へデータを移行する方法は2種類ございます。
    1. **GROWI.cloud サポートに移行を依頼する (推奨)**
       - 詳細につきましては、恐れ入りますが[お問い合わせフォーム](https://growi.cloud/v2/contact)からお問い合わせください。
    1. [**データアーカイブ・インポート機能を利用する**](/ja/cloud/data-migration/data-archive-import.html)
        - 上記のページに書かれた条件を満たしている場合のみご利用いただけます。


## その他Wikiサービスから

- GROWI以外にも、各種Wikiサービスからの移行実績がございます。詳細はお問い合わせください。
