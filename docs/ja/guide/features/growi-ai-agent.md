# GROWI AI Agent を使う

## GROWI AI Agent とは

GROWI AI Agent は、GROWI 内の情報をもとに質問へ回答する、エージェント駆動の AI アシスタントです。GROWI v8.0 で、目的に応じて自ら検索と推論を繰り返す仕組みへ全面的に刷新されました。

事前に学習させたページ群へ質問するのではなく、GROWI AI Agent が質問の内容を読み取り、GROWI 内を全文検索し、見つかったページを能動的に開いて読み込みながら回答を組み立てます。回答は質問と同じ言語で返されます。

::: warning
GROWI AI Agent を利用するには、管理者による AI 機能の有効化が必要です。左サイドバーの「GROWI AI Agent」アイコンがグレーアウトして押せない場合は、まだ有効化されていません。
:::

管理者向けに、AI 機能の有効化やプロバイダー・モデルの設定手順は [GROWI AI Agent のセットアップと管理](/ja/admin-guide/management-cookbook/growi-ai-agent.html) を参照してください。

## メッセージを送る

チャットウィンドウ下部の入力欄に質問を入力し、Enter キーを押すと送信されます。改行を入れたい場合は Shift + Enter を使います。

回答は、GROWI AI Agent が GROWI 内を検索して関連ページを読み込みながら生成します。そのため、表示までに少し時間がかかります。回答の生成中は、続けて新しいメッセージを送れません。

## モデルを選ぶ

入力欄の下にあるモデル選択メニューから、回答に使用する AI モデルを選べます。選択できるモデルは、管理者が許可したモデルの一覧です。プロバイダー（OpenAI / Anthropic / Google / Azure OpenAI）ごとにグループ分けして表示されます。

選んだモデルは記憶され、次回以降のチャットでも引き続き使用されます。会話の途中でモデルを切り替えることもできます。

## `@` でページを参照する

質問の中で特定のページを参照させたいときは、入力欄で `@` を入力します。`@` に続けてページ名を入力すると、閲覧権限のあるページが候補として表示されます。

<!-- TODO: スクリーンショットを挿入（@ 入力によるページ候補の表示） -->

- `@` は、行頭または空白の直後で入力すると有効になります。
- 候補は入力に応じて絞り込まれます。矢印キーで候補を移動し、Enter キーまたは Tab キーで確定します。Esc キーで候補を閉じます。
- 確定すると、入力欄に選んだページのパスが挿入されます。挿入されたパスはクリックでそのページを開けます。

候補には、自分が閲覧権限を持つページのみが表示されます。権限のないページは候補に出ません。

::: tip
`@` で参照したページは、その時点でページ本文がすべて渡されるわけではありません。GROWI AI Agent へはページのパスが手がかりとして渡され、GROWI AI Agent が必要と判断した場合にそのページを開いて読み込みます。
:::

## 回答を再生成・コピーする

直前の回答には、再試行とコピーの操作があります。

- **再試行**: 同じ質問で回答を作り直します。
- **コピー**: 回答の内容をクリップボードにコピーします。

## チャット履歴（スレッド）

チャットのやり取りはスレッドとして自動的に保存されます。過去のスレッドは、左サイドバーの「GROWI AI Agent」アイコンから開く AI パネルで確認できます。

- AI パネルの「最近の項目」に、過去のスレッドが新しい順に一覧表示されます。
- スレッドのタイトルは、最初のやり取りの内容から自動で生成されます。
- 一覧からスレッドを選ぶと、その会話を再開できます。
- スレッドにカーソルを合わせて表示されるごみ箱アイコンから、スレッドを削除できます。

スレッドは自分専用で、他のユーザーからは見えません。また、他のユーザーのスレッドは削除できません。

## 回答が途中で止まった場合

回答が最後まで生成されずに終了した場合は、回答の下に理由を示すメッセージが表示されます。主なケースは次のとおりです。

- 回答が最大長に達した
- 処理ステップの上限に達した
- コンテンツポリシーにより停止した
- エラーが発生した

## エラーが表示された場合

通信やモデルの問題で回答を生成できなかった場合は、エラーメッセージが表示されます。「再試行」から、もう一度回答の生成を試せます。

エラーが繰り返し発生する場合は、選択しているモデルを変更するか、管理者に AI 機能の設定状況を確認してもらってください。
