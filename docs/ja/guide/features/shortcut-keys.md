# キーボードショートカット

GROWIで使用できるキーボードショートカットの一覧です。

ショートカット一覧は、`Ctrl + /` でいつでも表示できます。

## グローバルショートカット

ページの閲覧中やエディターの使用中など、画面を問わず使用できるショートカットです。

| 操作 | キー |
|---|---|
| ショートカットヘルプの表示/非表示 | `Ctrl` + `/` |
| ページ作成 | `C` |
| ページ編集 | `E` |
| 検索 | `/` |

### イースターエッグ

| 操作 | キー |
|---|---|
| コントリビューターを表示 | `↑` `↑` `↓` `↓` `←` `→` `←` `→` `B` `A` |
| ミラーモード | `X` `X` `B` `B` `A` `Y` `A` `Y` `↓` `←` |

## エディターショートカット

エディターを開いている状態で使用できるショートカットです。

| 操作 | キー |
|---|---|
| エディター内検索 | `Ctrl` + `F` |
| 保存 | `Ctrl` + `S` |
| インデント | `Tab` |
| 左インデント | `Shift` + `Tab` |
| 行削除 | `Ctrl` + `Shift` + `K` |
| 行を挿入（コメント投稿） | `Ctrl` + `Enter` |
| 行の移動 | `Alt` + `↓` or `↑` |
| 行のコピー | `Alt` + `Shift` + `↓` or `↑` |
| 複数カーソル | `Ctrl` + `Alt` + `↓` or `↑`（もしくは `Alt` + クリック） |

## 書式設定（エディター）

| 操作 | キー |
|---|---|
| 太字 | `Ctrl` + `B` |
| 斜体 | `Ctrl` + `Shift` + `I` |
| 取り消し線 | `Ctrl` + `Shift` + `X` |
| コードテキスト | `Ctrl` + `Shift` + `C` |
| ハイパーリンク | `Ctrl` + `Shift` + `U` |

## 行の設定（エディター）

| 操作 | キー |
|---|---|
| 番号付きリスト | `Ctrl` + `Shift` + `7` |
| 箇条書きリスト | `Ctrl` + `Shift` + `8` |
| 引用 | `Ctrl` + `Shift` + `9` |
| コードブロック | `Ctrl` + `Alt` + `Shift` + `C` |
| コメントアウト | `Ctrl` + `/` |

## その他のエディターショートカット

ショートカットモーダルには表示されませんが、以下のショートカットも使用できます。

| 操作 | キー |
|---|---|
| 元に戻す | `Ctrl` + `Z` |
| やり直し | `Ctrl` + `Y`（もしくは `Ctrl` + `Shift` + `Z`） |
| すべて選択 | `Ctrl` + `A` |

## Vim モード

エディター設定で Vim キーマップを選択すると、Vim のキーバインドが使用できます。標準の Vim コマンドに加えて、以下の GROWI 固有のカスタマイズがあります。

| 操作 | キー |
|---|---|
| インサートモードを抜ける | `J` `J`（もしくは `J` `K`） |
| 保存 | `:w` |
| 太字 | `Ctrl` + `Shift` + `B`（※ デフォルトの `Ctrl` + `B` はページスクロールに割り当てられているため） |

:::tip
Vim モードでは `Ctrl` + `B` がページスクロールに使用されるため、太字のショートカットが `Ctrl` + `Shift` + `B` に変更されます。その他の書式設定・行の設定のショートカットはデフォルトと同じです。
:::

## Emacs モード

エディター設定で Emacs キーマップを選択すると、Emacs のキーバインドが使用できます。

### カーソル移動

| 操作 | キー |
|---|---|
| 前の文字 | `Ctrl` + `B` |
| 次の文字 | `Ctrl` + `F` |
| 前の行 | `Ctrl` + `P` |
| 次の行 | `Ctrl` + `N` |
| 行頭 | `Ctrl` + `A` |
| 行末 | `Ctrl` + `E` |
| 前の単語 | `Alt` + `B` |
| 次の単語 | `Alt` + `F` |
| ドキュメントの先頭 | `Shift` + `Alt` + `<` |
| ドキュメントの末尾 | `Shift` + `Alt` + `>` |
| 指定行へ移動 | `Alt` + `G` |

### テキスト編集

| 操作 | キー |
|---|---|
| 行末まで削除 | `Ctrl` + `K` |
| 次の文字を削除 | `Ctrl` + `D` |
| 次の単語を削除 | `Alt` + `D` |
| 前の単語を削除 | `Alt` + `Backspace` |
| 行を分割 | `Ctrl` + `O` |
| 文字を入れ替え | `Ctrl` + `T` |
| 大文字にする | `Alt` + `U` |
| 小文字にする | `Alt` + `L` |

### 選択・コピー・ペースト

| 操作 | キー |
|---|---|
| マーク設定 | `Ctrl` + `Space` |
| リージョンを切り取り | `Ctrl` + `W` |
| リージョンをコピー | `Alt` + `W` |
| ヤンク（貼り付け） | `Ctrl` + `Y` |
| ヤンクローテーション | `Alt` + `Y` |
| すべて選択 | `Ctrl` + `X` → `H` |

### 検索

| 操作 | キー |
|---|---|
| 検索 | `Ctrl` + `S` |
| 逆方向検索 | `Ctrl` + `R` |
| 置換 | `Shift` + `Alt` + `5` |

### 元に戻す・コメント

| 操作 | キー |
|---|---|
| 元に戻す | `Ctrl` + `/`（もしくは `Ctrl` + `Z`） |
| やり直し | `Shift` + `Ctrl` + `/`（もしくは `Shift` + `Ctrl` + `Z`） |
| コメントのトグル | `Alt` + `;` |
| キーボードマクロをキャンセル | `Ctrl` + `G` |

## VSCode モード

エディター設定で VSCode キーマップを選択すると、VSCode のキーバインドが使用できます。

### 編集

| 操作 | キー |
|---|---|
| 行の削除 | `Ctrl` + `Shift` + `K` |
| 行を下に移動 | `Alt` + `↓` |
| 行を上に移動 | `Alt` + `↑` |
| 行を下にコピー | `Alt` + `Shift` + `↓` |
| 行を上にコピー | `Alt` + `Shift` + `↑` |
| 行を選択 | `Ctrl` + `L` |
| インデント追加 | `Ctrl` + `]` |
| インデント削除 | `Ctrl` + `[` |

### 複数カーソル・選択

| 操作 | キー |
|---|---|
| カーソルを上に追加 | `Ctrl` + `Alt` + `↑` |
| カーソルを下に追加 | `Ctrl` + `Alt` + `↓` |
| 選択範囲の各行にカーソル追加 | `Shift` + `Alt` + `I` |
| 次の一致を選択 | `Ctrl` + `D` |
| 一致をすべて選択 | `Ctrl` + `Shift` + `L` |

### 検索・移動

| 操作 | キー |
|---|---|
| 検索 | `Ctrl` + `F` |
| 指定行へ移動 | `Ctrl` + `G` |
| 対応するブラケットへ移動 | `Ctrl` + `Shift` + `\` |

### コード折りたたみ

| 操作 | キー |
|---|---|
| 折りたたみ | `Ctrl` + `Shift` + `[` |
| 展開 | `Ctrl` + `Shift` + `]` |
| すべて折りたたみ | `Ctrl` + `K` → `Ctrl` + `0` |
| すべて展開 | `Ctrl` + `K` → `Ctrl` + `J` |

### コメント

| 操作 | キー |
| --- | --- |
| 行コメントのトグル | `Ctrl` + `/` |
| ブロックコメントのトグル | `Shift` + `Alt` + `A` |
| 行コメント追加 | `Ctrl` + `K` → `Ctrl` + `C` |
| 行コメント削除 | `Ctrl` + `K` → `Ctrl` + `U` |

### 元に戻す・補完

| 操作 | キー |
| --- | --- |
| 元に戻す | `Ctrl` + `Z` |
| やり直し | `Ctrl` + `Y`（もしくは `Ctrl` + `Shift` + `Z`） |
| すべて選択 | `Ctrl` + `A` |
| 補完の開始 | `Ctrl` + `Space` |
