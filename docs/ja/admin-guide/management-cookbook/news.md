# ニュース配信機能

ニュース配信機能とは、v7.5.4から追加された、GROWI 開発チームからのお知らせやリリース情報などのニュースを、GROWI 内のアプリ内通知から受け取れる機能です。

## 配信されるニュースの種類

配信されるニュースには以下の種類があり、種類によって表示対象が異なります。

| 種類 | 対象 |
| :--- | :--- |
| バージョンリリース | 管理者 |
| セキュリティパッチ | 管理者 |
| 新機能の紹介 (Tips) | 全ユーザー |
| メンテナンス告知 | 全ユーザー |
| 非推奨・EOL 通知 | 管理者 |

## 受信設定

管理者は `管理画面` > `アプリ設定` > `ニュース配信` からニュース配信を有効/無効に設定できます。
