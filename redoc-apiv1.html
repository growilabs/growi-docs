Remove page

Authorizations:
(beareraccessTokenInQuery)
Request Body schema: application/json
required
page_id
required
string <ObjectId>

Unique identifier of the page to delete

revision_id
string <ObjectId>

Revision ID for conflict detection

completely
boolean
Default: false

Whether to delete the page completely (true) or soft delete (false)

boolean
