# GROWI v5.1.x へのアップグレード

v5.1 では大きく以下の機能が実装されました。

  • 監査ログ
  • API のレート制限を管理者が設定できる機能

監査ログに関しては、環境変数を設定することで有効になるので利用する場合は 監査ログのセットアップ を参考に設定してください。

# アップグレード前にチェックすべきこと

  • GROWI の利用者に対して、今後監査ログが取得、保管されることが周知されているか
