# プラグイン

プラグインをインストールすることで GROWI をカスタマイズできます。 公開されているプラグインは GROWI プラグインサイト (opens new window)をご確認ください。

  • プラグインのインストール方法
  • プラグインインストーラー
  • プラグインカード

このページの内容は、動画でも解説しています。

# プラグインのインストール方法

  1. GROWIプラグインサイト (opens new window)にアクセスします。

  2. インストールするプラグインのリポジトリURLをコピーします。

  1. GROWI に管理者アカウントでログインし、「プラグイン」にアクセスします。
  1. リポジトリ URL を入力します。
  1. インストールボタンをクリックします。
  1. プラグインがプラグインカードに追加されます。また、ON/OFFを操作するとプラグインの有効化/無効化ができます。

# プラグインインストーラー

  • プラグインをインストールするには、GitHub リポジトリの URL を指定します。
  • リポジトリのブランチ指定も可能です。

WARNING

プラグインがすでにインストールされている場合、同じリポジトリから再度インストールするとプラグインは上書きされます。 また、ブランチが異なっていてもリポジトリが同じであれば上書きされます。

# プラグインカード

  • インストールしたプラグインを確認できます。
  • プラグインの有効化/無効化や削除が行えます。
