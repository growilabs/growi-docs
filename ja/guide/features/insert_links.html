# リンクを挿入する

# ツールバーのリンク挿入機能を使う

  1. 編集モードで、左下にあるツールバーの「＋」ボタンをクリックし、「Link」を選択します。

  2. リンク編集ダイアログが表示されます。リンク先: https://docs.growi.org/、ラベル: GROWI Docs と入力し、「完了」ボタンを押します。

  3. リンクが挿入されます。プレビューで確認してみましょう。

# Markdown で記述する

  • 以下のように編集画面に記述します。

    [Label](URL or page path)
