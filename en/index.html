Markdown x Wiki

Using the split-screen live preview, you can see what you are writing in real-time. You can use emojis, footnotes, create to-do lists, write raw HTML with Bootstrap, and syntax highlighting for various languages is provided.

Highly Customizable

Pick your own theme, authentication method, and keymap (Vim/Emacs/Sublime Text). Customize GROWI to satisfy the demands of both administrators and users.
